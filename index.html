Terms and Conditions

PAYMENT:

Payment through Paypal only
Payment must be received before I start working on colored version
Payment must be received within 5 business days after I give you the sketch version


FLOW OF WORK:

Step 1: Sketch - you can request changes to the drawing (up to 2 edits, NOT touching the main pose. For example: placement of hands or expression can be altered but the full pose not)
Step 2: Color plan / Rough colors - you can request changes to the base colors (the sketch cannot be touched at this point)
Step 3: Rendering and final image - no changes are possible at this stage


EXTRA FEES:

Reference images are required. If you need me to work from description alone, please don’t be surprised if it’s not 100% what you want.
In case of very complicated character designs, price may also raise by 10-20%. If this is the case, I will let you know in advance.


POSSIBLE REFUNDS:

If you decide to cancel BEFORE I start working on your request If I cannot provide/finish the artwork for any reason I can only provide a refund if any of the above conditions are met. No refunds in any other cases.


DEADLINES:

Artwork will be finished within 1-2 months after payment ( Fastest will be 1-2 weeks) varies on my disponibility If you need me to meet a certain deadline, let me know in advance and I’ll let you know whether I can work with those conditions or not


WHAT YOU GET:

Digital illustration (there is no physical copy)
Full resolution digital (2400 pixel per 2800 pixel if you don’t tell me the size that you want)
Output: high-quality JPEG (unless you request a different format)

Hell Girl: Ai Enma Interactive Experience

A web-based interactive project inspired by Hell Girl (Jigoku Shoujo). The experience lets users explore a dark, immersive world through multiple interactive interfaces, all styled like a phone app.

Key Features:

  • Interactive Chat with Ai Enma: A branching dialogue system where user choices influence the conversation.
  • Tarot Card Interface: A clickable tarot deck that reveals cards with animated effects, offering mystical readings.
  • Voodoo Doll Interaction: A mini-game allowing users to interact with a virtual voodoo doll, enhancing the dark, eerie atmosphere.
  • Immersive UI/UX: Animated backgrounds, phone-like app navigation, and responsive design for a cohesive, spooky aesthetic.
  • Modular Design: Each mini-experience is separated into reusable, maintainable modules for easy expansion.

Technologies: HTML, CSS, JavaScript, DOM manipulation, animation effects.

This project highlights front-end development, interactive storytelling, and creative UI design.

A psychological, single-character visual novel made in Ren’Py. The art is all done by me.
